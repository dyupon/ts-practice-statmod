\section{Определение сильной и слабой разделимости при смешивании компонент}

Определить, что компоненты смешались, можно с помощью матрицы взвешенных корреляций.

Отсутствие сильной разделимости (при наличии слабой) может быть вызвано следующими факторами:
\begin{itemize}
\item Собственные числа компонент слишком близки (для определения можно воспользоваться графиком собственных чисел).
\end{itemize}

Отсутствие слабой разделимости может быть вызвано следующими факторами:
\begin{itemize}
\item Ряд слишком короткий (не хватает длины ряда для сходимости)
\item Ряд слишком зашумлён (компоненты шума смешиваются с компонентами сигнала)
\end{itemize}
