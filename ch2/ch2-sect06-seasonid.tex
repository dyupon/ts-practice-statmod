\section{Идентификация периодичности (сезонности)}

Для определения сезонности можно воспользоваться:
\begin{enumerate}
\item Графиком собственных чисел (ступеньки из двух точек на одинаковом уровне)
\item Графиком собственных векторов (гармонические компоненты)
\item Графиком пар собственных векторов (образуют правильные многоугольники с количеством вершин, соответствующим периоду)
\item Графиком взвешенных корреляций (пары последовательных компонент, коррелирующие друг с другом)
\end{enumerate}

Отдельный случай: частота $\displaystyle\frac{1}{2}$.
