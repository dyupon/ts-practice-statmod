\section{Элементарные восстановленные компоненты}

По графикам элементарных восстановленных компонент можно также пытаться определять:
\begin{itemize}
\item Компоненты тренда (медленно меняющиеся компоненты)
\item Пилообразную компоненту, соответствующую частоте $\displaystyle\frac{1}{2}$.
\item Гармонические компоненты (пары похожих по поведению гармонических элементарных компонент) и то, не смешались ли они (при смешении двух компонент с близкими частотами образуются <<виолончели>> --- компоненты, огибающие которых тоже являются гармоническими компонентами)
\end{itemize}
