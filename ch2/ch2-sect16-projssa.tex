\section{Projection SSA, выделение линейного тренда}

Идея SSA с проекциями заключается в следующем: рассмотрим временной ряд $\tsX$ длины $N$ и его траекторную матрицу $\mtX$. Тогда схема метода SSA with projection может быть описана следующим образом:

\begin{itemize}
\item Построим матрицу $\mtC = \mtC_{\mtX}$, основываясь на некоторой априорной информации о временном ряде $\tsX$
\item Вычислим $\mtX^{\prime} = \mtX - \mtC$
\item Проведём SVD-разложение матрицы $\mtX^{\prime} = \sum\limits_{i} \sqrt{\lambda_{i}^{\prime}}U^{\prime}_{i}V_{i}^{\prime \mathsf{T}}$
\end{itemize}

В результате получим разложение матрицы $\mtX$: $\mtX = \mtC + \mtX^{\prime} = \mtC + \sum\limits_{i} \sqrt{\lambda_{i}^{\prime}}U^{\prime}_{i}V_{i}^{\prime \mathsf{T}}$

Особенность метода заключается в способе построения матрицы $\mtC$. Если в качестве матрицы $\mtC$ использовать проектор на пространство, порождённое вектором из единиц, получим SSA с однократным центрированием (по строкам или по столбцам). Таким же образом можно получить SSA с двукратным центрированием (как по строкам, так и по столбцам). Данные применения хороши для выделения константного и линейного трендов. В общем случае можно строить проектор исходя из дополнительной информации о структуре сигнала, например, если имеется ряд с сигналом похожей структуры.
