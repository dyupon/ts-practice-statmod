\section{MSSA для анализа многомерных временных рядов}

Метод MSSA является обобщением метода SSA на случай многомерных временных рядов.

Основные отличия метода MSSA от базового метода SSA:
\begin{enumerate}
\item На шаге вложения траекторная матрица ряда составляется из траекторных матриц его компонент:
\begin{equation*}
\mtX = \opT_{\text{MSSA}}^{(L)}(\tsX) = \left[\opT_{\text{SSA}}^{(L)}(\tsX^{(1)}):\ldots:\opT_{\text{SSA}}^{(L)}(\tsX^{(s)}) \right] = \left[ {\mtX^{1}}:\ldots:{\mtX^{(s)}} \right].
\end{equation*}
\item На шаге диагонального усреднения матрицы $\mathbf{X}_{I_{j}}$ приводятся к составному ганкелеву виду с помощью покомпонентного применения оператора ганкелизации:
\begin{equation*}
\overline{\opH}(\mtX_{I_{j}}) = \left[ \opH(\mtX_{I_{j}}^{(1)}):\ldots:\opH(\mtX_{I_{j}}^{(s)}) \right].
\end{equation*}
\end{enumerate}

Метод MSSA может дать результаты лучше, чем применение базового SSA к обоим рядам, если у рядов есть компоненты с общей структурой (т.е. лежащие в одном подпространстве пространства, образованного траекторной матрицей).
