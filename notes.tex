\chapter{Общая часть}

\begin{itemize}
\item Что такое периодограмма?
\item Что такое тренд (разные варианты определения)? Как выглядит периодограмма тренда?
\item Что такое периодическая компонента? Как выглядит ее периодограмма?
\item Что такое шум? Как выглядит его периодограмма?
\item Периодограмма как оценка спектральной плотности. Распределение значений. Сглаживание периодограммы.
\item Чем отличается сглаживание от выделения тренда?
\item Что такое линейный фильтр, импульсная характеристика? Причинный фильтр, FIR.
\item Характеристики фильтра через его воздействие на $\cos(2 \pi \omega n)$ (или на комплексную экспоненту) – АЧХ, ФЧХ.
\item AЧХ фильтра скользящего среднего, зависимость от длины окна.
\item АЧХ фильтра перехода к разностям (дифференцирования).
\item Что такое запаздывание и отчего оно может возникать (на примере скользящего среднего)?
\item Смещение при сглаживании фильтром скользящего среднего. Роль второй производной.
\item Фильтр для подавления шума. Роль нормы коэффициентов фильтра.
\item Как связаны периодограммы ряда до применения фильтра и после применения фильтра?
\item Модели данных – аддитивная и мультипликативная.
\item Методы стабилизации дисперсии в разных моделях (логарифмирование, извлечение квадратного корня, …).
\item Выделение тренда у ряда с сезонностью (выбор длины окна в скользящем среднем).
\item Переход к разностям – плюсы и минусы (устранение тренда, превращение ряда в стационарный, усиление вклада высоких частот).
\item Скользящее среднее и скользящая медиана.
\item Растекание частоты в периодограмме. Подправка длины ряда для ее устранения.
\item  Выделение тренда с помощью параметрической регрессии.
\item Выделение тренда с помощью метода LOESS
\item Нахождение огибающей периодического ряда с помощью выделения тренда.
\item Оценивание поведения дисперсии шума с помощью выделения тренда.
\item Метод разложения Classical seasonal decomposition.
\item Метод разложения STL.
\end{itemize}

\chapter{Метод SSA}

\begin{itemize}
\item Как выбирать L.
\item Последовательный SSA.
\item Слабая и сильная разделимость.
\item Компоненты смешались. Как понять, это слабая или сильная разделимость?
\item Как идентифицировать тренд?
\item Как идентифицировать периодичность?
\item Использование матрицы взвешенных корреляций.
\item Элементарные восстановленные компоненты.
\item Корни хар.полинома, сигнальные и лишние.
\item Оценка параметров в SSA.
\item Прогноз.
\item Доверительные интервалы.
\item Автоматическая идентификация.
\item Заполнение пропусков.
\item Теплицев SSA для стац. рядов.
\item Projection SSA, выделение лин.тренда.
\item Улучшение разделимости с помощью вращения в выбранном подпространстве: Iterative O-SSA (слабая и сильная разделимость) и DerivSSA (сильная разделимость).
\item Аппроксимация Cadzow SSA.
\item MSSA для анализа многомерных временных рядов. Когда лучше анализировать ряды вместе, а когда отдельно?
\item 2D-SSA для разложения изображения.
\end{itemize}
